Flexbox

This is a lil' description of Flexbox

Order Properties

Order properties determine how the order of items are laid out despite how they appear in the markup. Assign ordering to items using the classes xs-flex-order-1, .xs-flex-order-2, or .xs-flex-order-3.

.xs-flex-order-1 .md-flex-order-3 First item in the markup
.xs-flex-order-2.md-flex-order-2 Second item in the markup
.xs-flex-order-3 .md-flex-order-1 Third item in the markup
{% highlight html %}
.xs-flex-order-1 .md-flex-order-3
.xs-flex-order-2
.xs-flex-order-3 .md-flex-order-1
{% endhighlight %}

Direction

This applies to the direction that items are placed along the main axis. Assigning a parent .xs-flex-row means that the children will be distributed left to right. A left to right direction on the x-axis is also the .display-flex default.
